# ぼくの島 (My Island)

島の大きさを地図上で可視化するWebアプリケーションです。

## 機能

- 淡路島・小豆島のオーバーレイ表示
- Google Maps上での任意の場所への配置
- クリックで表示/非表示の切り替え

## 使い方

1. 島を選択（淡路島・小豆島）
2. 地図上の任意の場所をクリック
3. 島のオーバーレイが表示される

## 開発

```bash
npm install
npm run watch  # 開発サーバー起動
npm run build  # ビルド
```
